# mimamori-kids-reserve
みまもりキッズの利用予約入力を一括でできるようにする Chrome 拡張

### Before

<img width="626" alt="image" src="https://user-images.githubusercontent.com/10782053/230921716-168879c0-d797-4d21-a255-d2bc4df74463.png">

### After 

「まとめて設定」の行が追加され、一括で全ての 利用希望・入室予定・退室予定・お迎え を設定できるようになります。

例えば、退室予定を16:30に設定すると、表示されている月の全ての退室予定が16:30に設定されます。

<img width="626" alt="image" src="https://user-images.githubusercontent.com/10782053/230923854-c4e42b87-7d57-423f-9720-b9a6198a6f04.png">

## 利用方法
1. 本リポジトリをローカルにダウンロード
2. Chrome で chrome://extensions/ にアクセスし「デベロッパーモード」を有効にする
3. 「パッケージ化されていない拡張機能を読み込む」をクリックし、ダウンロードした本リポジトリを読み込む
